# BAEquipDrop

___

基于HoshinoBot实现的碧蓝档案装备掉落分布查询插件，主要为了查询一些不显示的副掉落，方便在刷高级设计图时顺便补充缺失的低级素材。

**查询格式**

```
查掉落 t[1234567]+[装备种类]
```

**装备种类名**

帽子|手套|鞋子|背包|徽章|符咒|发卡|手表

___
